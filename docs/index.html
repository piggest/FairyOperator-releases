FairyOperator

macOS の操作を、妖精が手伝う。

キー入れ替え・ウインドウ操作・Dock 拡張・スリープ抑止をまとめた menubar ユーティリティ。

最新版をダウンロード GitHub

macOS 13 Ventura 以降 / Apple Silicon & Intel

妖精たち

  • キー入れ替え

    外付けキーボードごとに任意のキーを再マッピング。HID デバイス単位で制御。

  • アプリ切り替え

    Cmd+Tab を拡張。Unity プロジェクトのアイコンも自動で見つけて表示。

  • クリップボード

    履歴を保持し、すばやく検索・呼び出し。

  • ウインドウ操作

    ホットキーで分割・最大化・移動。

  • IME 操作

    日本語入力のオン/オフを意のままに。

  • メニューバー

    ステータスバーから機能を即時切替。

  • Dock 拡張

    応答なしのアプリに砂時計、ウインドウ数バッジなど Dock アイコンを賢く。

  • スリープ抑止

    必要なときだけ自動でスリープを止める。

スクリーンショット

インストール

  1. 最新リリース から FairyOperator.app.zip をダウンロード。
  2. 解凍して /Applications に配置。
  3. 初回起動時にアクセシビリティ権限と入力モニタリング権限を許可。

Developer ID 署名 + Notarization 済み。Gatekeeper 警告は出ない想定。
