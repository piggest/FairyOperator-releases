FairyOperator

Fairies that help you control macOS.

A small toolbox of fairies that lend a hand with everyday macOS tasks.

Download Latest Feature Guide

macOS 13 Ventura or later / Apple Silicon & Intel

The Fairies

  • Key Remap

    Remap any key on a per-keyboard basis. Profiles are managed per HID device.

  • App Switcher

    A more flexible switcher than Cmd+Tab. Detects and shows Unity project icons automatically.

  • Clipboard

    Keeps a history of copied items with fast search and quick recall.

  • Window Manager

    Split, maximize, or move windows between displays with a single hotkey.

  • IME Control

    Toggle Japanese input on/off with any key, customizable per app.

  • Menu Bar

    Temporarily hides the app menu so more status icons fit on screen.

  • Dock Extensions

    Adds badges to Dock icons — hourglass for not-responding apps, window count, and more.

  • Sleepless

    Keeps macOS awake only when you need it, and restores the original state automatically.

Install

  1. Download FairyOperator.app.zip from the latest release.
  2. Unzip and move the app into /Applications.
  3. On first launch, grant Accessibility and Input Monitoring permissions.

Signed with a Developer ID and notarized by Apple, so Gatekeeper will not show a warning.
